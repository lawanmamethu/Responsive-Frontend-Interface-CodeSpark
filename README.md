## 📖 About The Project

**CodeSpark** is a modern, fully responsive frontend interface built as Project 1 for DecodeLabs Industrial Training Program. This project demonstrates mastery of **mobile-first design**, **semantic HTML5**, **CSS Grid & Flexbox**, and **interactive JavaScript**.

The design follows **2025 UI/UX trends** with warm, grounded aesthetics and strict accessibility standards.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 📱 **Mobile-First Design** | Optimized for all screen sizes |
| 🌙 **Dark Mode Toggle** | Smooth light/dark theme switching |
| 💎 **Premium UI** | Clean, modern, professional layout |
| 🎨 **Custom Color Scheme** | Mocha Mousse (#F2F0EA) + Ethereal Blue |
| ♿ **Accessible** | Semantic HTML + WCAG compliant |
| ⚡ **Interactive Elements** | Hover effects, smooth transitions |
| 📄 **Multi-Page Structure** | 12+ interconnected pages |
| 🎭 **Emoji Rain Effect** | Fun, dynamic visual experience |
